How to run this project

First run the notebook to save the cnn_model.h5 from the zip file present in the repo then follow the steps below.

1) Create a virtual environment
2) Open Gitbash/CMD/Powershell
3) Write python -m venv virtualenv
4) For gitbash -> source virtualenv/Scripts/activate
5) For CMD/Powershell -> cd virtualenv/Scripts/activate -> cd ../../
6) Then while staying in virtual environment write -> pip install Flask tensorflow numpy
7) Run the app by writing python main.py
